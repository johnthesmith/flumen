# Дисклеймер

Это игровая песочница, а не теория всего. Мы исследуем, какие структуры 
возникают из плоского списка с минимальными правилами. Физические термины 
(время, пространство, метрика и иное) используются только для удобства и не 
претендуют на описание реальности.

Так же см [О бесконечности в песочнице](./inf.md).




# Что такое флюмен?

[Флюмен](./flumen.md) (лат. *flumen* — поток) — абстрактная однонаправленная 
связь между двумя метками. Флюмен не имеет координат и не вложен в пространство. 
Флюмены — базовые элементы, из которых строятся все события и сущности включая 
[внутреннего наблюдателя](./observers.md).



# Пространства и флюмены

Из флюменов формируются списки. При интерпритации связей могут быть выявлены 
"пространственные" структуры. Например, несколько флюменов могут образовывать 
замкнутые (компактные) или открытые многомерные "пространства" и "временные" 
лини.


# Как работают флюмены?

Флюмены неподвижны. Однако они пропускают через себя [информацию](./information.md). Флюмены 
можно рассматривать как каналы или пути.



## Применение и перспектива

Флюмен теория может быть полезна для создания моделей пространства из простых 
списков из которых следуют эмерджентные свойства.
