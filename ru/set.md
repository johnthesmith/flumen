# Множество флюменов

Состояние системы — конечное множество флюменов. Отображается плоским списком, 
содержащим перечень кортежей флюменов. Других исходных состояния нет. 

Обнозначается `𝔽` или `F`.

```
𝔽 = { f₁[a₁, i₁, b₁], f₂[a₂, i₂, b₂], …, fₙ[aₙ, aₙ, bₙ] }
```



# Свойства множества

1. Замкнутость — множество содержит все рассматриваемые флюмены.
0. Дискретность — множество флюменов конечно.

> ⚠️ Важно! Для целей анализа множеству флюменов сопоставляется [графа 
> флюменов](./flumen_graph.md). Это лишь способ представления. Онтологически 
> состояние системы — плоский список пар меток. Никакая геометрия при описании 
> множества не предполагается. Пути, связность и иные структуры возникают только 
> как операции над метками.



# Операции над множествами

Операции над множествами флюменов соответствуют стандартным множественным 
операциям, но с учетом того, что идентификация флюмена происходит по его 
[меткам](./label.md) `[a, b]`, а не по уникальному идентификатору.

Над множествами флюменов можно выполнять следующие операции:
1. [Добавление](#добавление)
0. [Удаленеи](#удаление)



## Добавление

Добавление множества флюменов `F₂` в множество `F₁` — операция, при которой все 
флюмены из `F₂` включаются в `F₁`:

```
F₁ := F₁ ∪ F₂
```

При совпадении меток `[a, b]` флюмен сохраняется в единственном экземпляре. 
Операция изменяет исходное множество `F₁`, дополняя его новыми флюменами.


## Удаление

Удаление множества флюменов `F₂` из множества `F₁` — операция, при которой из 
`F₁` исключаются все флюмены, присутствующие в `F₂`:

```
F₁ := F₁ \ F₂
```

Идентификация флюмена происходит по полному совпадению пары меток 
`[a, b]`. Если в `F₂` есть флюмен `[a, b]`, из `F₁` будут удалены **все** 
флюмены с такой же парой меток (если их несколько). Операция изменяет исходное 
множество `F₁`.
