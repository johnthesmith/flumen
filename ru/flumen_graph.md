# Граф флюменов

Граф флюменов — ориентированный граф, порожденный [множеством 
флюменов](./set.md) как способ его представления.



# Формально

Граф флюменов G(F) индуцируется множеством F.
Вершины = метки из кортежей (a,b)
Ребра = флюмены (a→b)



# Статус

Граф вторичен и эмерджентен — это интерпретация, а не исходная сущность. 
Первичен всегда плоский список флюменов. Термин используется для удобства 
анализа (пути, связность, компоненты).



# Свойства

1. Индуцированность — граф полностью определяется множеством флюменов F: вершины
извлекаются из кортежей, ребра соответствуют флюменам.
2. Ориентированность — каждое ребро имеет направление от начальной метки к 
конечной.
3. Дискретность — число вершин и ребер конечно (в рамках рассматриваемой 
модели).
4. Возможная несвязность — граф может распадаться на несколько компонент 
связности (в неориентированном смысле), если между подмножествами меток нет 
соединяющих флюменов.
5. Отсутствие геометрии — граф задает только комбинаторную структуру связей; 
метрические и координатные свойства не предполагаются.
